Principais elementos:

  • Títulos: <h1> até <h6>
  • Parágrafos: <p>
  • Listas não ordenadas: <ul>
  • Listas ordenadas: <ol>

Lista de tarefas

  1. Abrir o VS Code ou Bloco de Notas
  2. Criar um novo arquivo HTML
  3. Adicionar elementos como Título, parágrafos e listas
  4. Salvar e abrir no navegador usando Go Live
